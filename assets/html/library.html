Java Library Program

The Online Library is a local REST-Web-Service Application with JSON. I have used JavaFX for the client and MySQL for the Database. The Program is made to work with multiple clients at the same time and allows users to rent and return books and admins to add, delete or edit books. I’ve completed this project during my programming 3 course in my 3rd semester.
