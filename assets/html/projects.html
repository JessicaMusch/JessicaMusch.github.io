Between Dice and Madness: A classic Adventure

Between Dice and Madness: A classic Adventure is a short movie about a classic Dungeons & Dragons session with all the chaos and madness. The movie is shot partially in real life and partially with the help of Motion Capture in 3D. The real life scenes depict the table with the players and the Dungeon Master, while the animated scenes purely show the vision of the characters in the mind of the players.

Java Library Program

The Library is a local REST-Web-Services application with JSON and JDBC access. The application allows the user to create an account and rent and return books in the library. Admins can add new books to the library, edit existing ones or delete them. Multiple Clients can run at the same time.
